# MERN Todo App (CRUD)

This project was originally built as a hiring task for IntelliGlanceTech, where the requirement was to create a basic CRUD application using the MERN stack.

At the time, I was still getting comfortable with full stack development, but I ended up enjoying the process and decided to go a bit beyond the initial scope. I added extra features like filtering, pagination, and better structure on both the frontend and backend to make it feel closer to a real application.

## 🧠 What This Project Shows

This project represents my early foundation in:
- Building full stack applications using the MERN stack
- Understanding CRUD operations end-to-end
- Connecting frontend UI with backend APIs
- Structuring a backend with authentication and protected routes
- Managing application state on the frontend

It’s not meant to be a polished production app, but rather a solid representation of where I started getting comfortable working across the full stack.

- User authentication (register/login with JWT)
- Protected routes
- Create, read, update, delete items
- Item filtering (category, visibility, completion state)
- Pagination support
- Public vs private item visibility
- Basic API testing setup

## 🛠️ Running Locally

### 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git  
cd YOUR_REPO_NAME  

### 2. Install dependencies
cd backend  
npm install  

cd ../frontend  
npm install  

### 3. Setup environment variables

Create a `.env` file in the backend:

MONGO_URI=your_mongodb_connection  
JWT_SECRET=your_secret  
PORT=5000  

### 4. Run the app

Backend:  
npm run dev  

Frontend:  
npm start
